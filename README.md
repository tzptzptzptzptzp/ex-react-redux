## 紹介

こちらは React を使用したカウンターアプリです。  
Redux を使用して状態管理をしています。

## 使用技術

フロントエンド

- [React](https://ja.react.dev/)
- [Redux](https://redux.js.org/)
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwindcss](https://tailwindcss.com/)

## 開発手順

npm を使用しています

```bash
npm
npm run dev // 開発用サーバー起動
```
